# 🧩 DSA Patterns

## HashMap Pattern

### Clues

- Frequency count
- Duplicate detection
- Pair lookup

### Questions

- Two Sum
- Valid Anagram
- Contains Duplicate

### Complexity

Time: O(n)
Space: O(n)

## Binary Search Pattern

### Clues

- Sorted array
- Search space
- Find first/last occurrence

### Complexity

Time: O(log n)
Space: O(1)
